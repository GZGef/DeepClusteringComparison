```

   ‚îî‚îÄ> –≠–Ω–∫–æ–¥–µ—Ä: X ‚Üí Z (—ç–º–±–µ–¥–¥–∏–Ω–≥–∏)

   ‚îî‚îÄ> Soft assignment: Q = f(Z, Œº)  # –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–Ω–æ—Å—Ç–∏
   ‚îî‚îÄ> Target distribution: P = g(Q)  # –¶–µ–ª–µ–≤–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ

### –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ —Å–µ—Ç–∏:

```

   Gender]   [16]                   [16]        Gender]

### –§–æ—Ä–º—É–ª—ã DEC:

**Soft assignment:**
```

```

**Target distribution:**
```

```

**Loss function:**
```

```
